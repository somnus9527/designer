## Page Designer Monorepo
> 界面设计器仓库, 设计器基于 !(chameleon)[https://github.com/hlerenow/chameleon]

### Monorepo结构

```
page-designer
├── README.md
├── apps  -- 应用目录
│   ├── base  -- 基座应用
│   └── core  -- 设计器核心包
├── libs  -- 工具包目录
│   ├── materials  -- 物料目录
│   ├── types  -- 类型定义应用
│   └── utils  -- 通用工具应用
├── package.json
├── pnpm-lock.yaml
└── pnpm-workspace.yaml
```
